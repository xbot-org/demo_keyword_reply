# demo_keyword_reply

基于 xbot 实现微信关键词自动回复

## 使用

先配置好 xbot 的配置如下，并运行
```
CALLBACK_URL=http://127.0.0.1:5678
SECRET=xxx
PORT=5557
```

clone 代码
```
git clone https://github.com/xbot-org/demo_keyword_reply.git
cd demo_keyword_reply
```

运行此服务 `php -S localhost:5678`
